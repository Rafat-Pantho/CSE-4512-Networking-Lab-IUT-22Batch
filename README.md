# CSE-4512 Networking Lab (IUT 22 Batch)

This repository contains materials for the Networking Lab (Course: CSE-4512) for the IUT 22 Batch.

## Contents
- `lab 1/220041102_lab01.pkt` : Cisco Packet Tracer lab file for Lab 1.
- `lab 1/lab01_text.txt` : Supporting notes / instructions for Lab 1.
- `lab 1/eng.traineddata` : OCR trained data (if used for extracting text from images or screenshots in the lab).

## Getting Started
1. Install Cisco Packet Tracer (matching the version used in class if possible).
2. Clone this repository:
   ```bash
   git clone https://github.com/Rafat-Pantho/CSE-4512-Networking-Lab-IUT-22Batch.git
   ```
3. Open the `.pkt` file in Packet Tracer.
